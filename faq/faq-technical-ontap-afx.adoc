.What are the supported data protocols in disaggregated ONTAP?

The data protocols supported in disaggregated ONTAP include the following:

* NFSv3, NFSv4.0, NFSv4.1, NFSv4.2 
* SMB2.x, SMB3.x
* S3
* NDMP
