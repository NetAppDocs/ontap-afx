== Autonomous Ransomware Protection with AI (ARP/AI) enhancements

[cols="2,4" options="header"]
|===
// header row
| Update
| Description

| ARP/AI support for eligible SnapMirror primary volumes
a| Beginning in ONTAP 9.19.1 RC, ARP/AI supports eligible primary volumes in SnapMirror Synchronous relationships on supported AFX configurations. ARP/AI runs on the active read-write primary volume, creates primary-only ARP snapshots, and provides machine-learning-based ransomware detection.

// table end
